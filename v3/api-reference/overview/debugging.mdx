## `countries` format
You should use [ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) code. 

For example, France - `FR`

You can find the entire list on the [ISO standards website](https://www.iso.org/obp/ui/#search)
